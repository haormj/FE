###1.首先写写出一级目录

    <style type="text/css">
    	ul{
    		/*去掉ul的默认样式*/
    		margin:0px;
    		padding:0px;
    		list-style: none;
    		/*为了解决子元素float时,父元素大小为0的问题*/
    		overflow: hidden;
    	}
    	li{
    		float:left;
    	}
    	a{
    		/*让a标签作为块显示,而不会只是选中文字*/
    		display:block;
    		width:60px;
    		text-align: center;
    	}
    	a:hover{
    		background-color: red;
    	}
    </style>
    <ul>
    	<li><a href="#">菜单一</a></li>
    	<li><a href="#">菜单二</a></li>
    	<li><a href="#">菜单三</a></li>
    	<li><a href="#">菜单四</a></li>
    </ul>

上面的这段代码实现了最简单的菜单,大家直接就能够看懂
###2.要想实现二级子菜单,我自己的方式就是首先做出静态的效果,然后在将其隐藏,最后通过js结合事件来实现最后的效果.
下面是具体的代码,也是上面代码的改良版.

    <style type="text/css">
    	.fld{
    		/*去掉ul的默认样式*/
    		margin:0px;
    		padding:0px;
    		list-style: none;
    		/*为了解决子元素float时,父元素大小为0的问题*/
    		overflow: hidden;
    	}
    	.fld > li{
    		float:left;
    		width:70px;
    	}
    	.fld > li > a{
    		/*让a标签作为块显示,而不会只是选中文字*/
    		display:block;
    		width:60px;
    		text-align: center;
    	}
    	.fld > li > a:hover{
    		background-color: red;
    	}
    	.sld {
    		margin:0px;
    		padding:0px;
    		list-style:none;
    		display:none;
    	}
    </style>
    <script type="text/javascript">
    	/*对js不是很熟悉*/
    	function showMenu(current){
    		current.childNodes[1].nextElementSibling.style.display = "block";
    	}
    	function hideMenu(current){
    		current.childNodes[1].nextElementSibling.style.display = "none";
    	}
    </script>
    <ul class="fld">
    	<li onmouseover="showMenu(this)" onmouseout="hideMenu(this)"><a href="#" >菜单一</a>
    		<ul class="sld">
    			<li><a href="#">子菜单一</a></li>
    			<li><a href="#">子菜单二</a></li>
    		</ul>
    	</li>
    	<li><a href="#">菜单二</a></li>
    	<li><a href="#">菜单三</a></li>
    	<li><a href="#">菜单四</a></li>
    </ul>
